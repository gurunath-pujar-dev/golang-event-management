# Event-Management-App

A web application for management of in-person and live events.

---

## ✨ Features

- Dashboard for all events
- Updates in real-time
- Add attendees
- Update events
- JWT Auth

---

## 🛠 Tech Stack

- ReactJS - SPA frontend
- Tailwind CSS - Styling (via CDN)
- Gin - Fast HTTP web framework
- Sqlite3 - Database 

---


## ⚙️ Execution steps:

Backend:

Clone & Install dependencies:

```bash
git clone https://github.com/gurunath-pujar-dev/golang-event-management.git
cd golang-event-management/api
go mod tidy
```

Run command:
```bash
go run ./cmd/api/*.go
```

---
Frontend:

Install dependencies:

```bash
cd golang-event-management/client
npm install
```

Run command:
```bash
go run dev
```
---
